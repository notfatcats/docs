Let's start configuring the ban appeal module in your server. Before you begin, ensure you have configured your settings.


1. Configure your ban appeal module by executing the command:
```
/configure ban-appeal
```

2. Once you run the command, there will be 5 fields that you need to fill in. If you're unsure about what these fields do, you can watch the video below for a detailed explanation.

[Video: Ban Appeal Module Configuration Fields](https://760854805-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FTOO5C5f4DykDjLxxWgUD%2Fuploads%2FcjnJXkExOz3bMMzr6Etu%2F2023-12-08%2017-20-42.mp4?alt=media&token=efa556a2-5440-4f5a-9f0f-fb984a3f8aa6)
